How to Install Capture Website
Click on the green Code button at the top of this repository and select Download ZIP.

Extract the downloaded ZIP file to a permanent location on your computer.

Open your browser and navigate to chrome://extensions/ (or edge://extensions/ for Edge).

Turn on the Developer mode toggle in the top right corner.

Click the Load unpacked button in the top left.

Select the extracted folder containing the extension files.

The extension is now installed. Pin it from the extensions menu to start using it.

Enjoy!
